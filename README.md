# Risk-Constrained Reinforcement Learning for Financial Portfolio Optimization

## Overview
This project explores the application of Reinforcement Learning (RL) for financial portfolio optimization under real-world risk constraints. The system trains RL agents to dynamically allocate portfolio weights while balancing profitability and financial risk.

The project focuses on developing reliable and risk-aware AI systems for intelligent financial decision-making.

---

## Research Focus
This work investigates reliable reinforcement learning systems capable of operating under uncertainty while maintaining stable and risk-aware portfolio strategies. The project combines reinforcement learning, financial risk management, and trustworthy AI principles.

---

## Features

- Reinforcement Learning-based portfolio optimization
- Risk-aware reward engineering
- Portfolio allocation under financial constraints
- Interactive dashboard for performance visualization
- Risk-adjusted portfolio evaluation

---

## Reinforcement Learning Algorithms

Implemented and evaluated:
- A2C (Advantage Actor-Critic)
- REINFORCE Policy Gradient

---

## Risk Metrics

The portfolio strategies were evaluated using:

- Sharpe Ratio
- Sortino Ratio
- Conditional Value at Risk (CVaR)
- Drawdown Analysis
